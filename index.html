OOP

Classes

A class can be seen as a template for objects and is defined as followed:

Syntax
class ClassName {

}
Example: Creating a class
class Person {

}

A class name should be written in UppercaseCamelCase. A good practice is to have one class in each file and name the file the same as the class.

Properties

A class can hold different properties. These can be seen as the characteristics of a class. An example of a class with properties

Example: Class with properties
class Person {

public $name;
public $lastName;
public $age;

}

Objects

After defining a class it's possible to instantiate one or more objects from that class.

Syntax: Instantiate an Object
$newObject = new ClassName();
Example: Instantiate an Object from the class 'Person' and save it to the variable $myFriend
$myFried = new Person();

So if a class can be seen as a template, an instance of a class (also called object) can be seen as a copy of the template. After instantiating a class it's possible to fill out the template with unique data. In the case above, the class contains the property's name and age. These can be filled with data for the new $myFried object as followed:

Example: Set data on a object
$myFriend->name = 'Lucas';
$myFriend->lastName = 'Bloom';
$myFriend->age = 30;
Example: Access the data saved on an object
echo $myFriend->name . ' is ' . $myFriend->age = 30 . ' years old!';
prints: Lucas is 30 years old!
Methods

Classes can also hold methods. Every object instantiated from the class can use these methods. A method is defined as follewed:

Syntax
class ClassName {

public function functionName(<parameterOne>, <parameterTwo>){
//DO SOMETHING
}

}

In the example beneath the class 'Person' now contains a method. Inside methods, is it possible to access properties that are defined in the class by using the keyword $this.

Example: Class with properties and one method that prints the full name of the object
class Person {

public $name;
public $lastName;
public $age;

public function printFullName(<parameterOne>, <parameterTwo>){
print $this->name . ' ' . $this->lastName;
}

}

Accessing a method on an object is almost identical to accessing a property. To access a method an arrow is used followed by the name of the method and then an opening and closing parenthesis ().

Example: Access a method on an object
$myFriend->printFullName();
prints: Lucas Bloom
